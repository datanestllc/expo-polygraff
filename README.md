# ПолиграфФ — Expo Landing Page

Промо-страница типографии **ПолиграфФ** для выставок и мероприятий.

Производство самоклеящихся этикеток в рулонах методом флексографической печати.

## Что на странице

- Ключевые преимущества и цифры компании
- Перечень услуг (печать, премиальная отделка, «Честный знак»)
- Контакты менеджера (телефон, email, WhatsApp, сохранение vCard)

## Запуск

Статический сайт без зависимостей. Открыть `index.html` в браузере или запустить любой HTTP-сервер:

```bash
python3 -m http.server
```

## Деплой

GitHub Pages — автоматический деплой при пуше в `main`.

**Сайт:** https://datanestllc.github.io/expo-polygraff/

## Структура

```
index.html          — вся страница (HTML + CSS + JS)
qr/                 — QR-коды (vCard, ссылка на сайт)
.github/workflows/  — деплой на GitHub Pages
```
